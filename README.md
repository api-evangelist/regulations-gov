# Regulations.govThis is a repo for managing machine readable artifacts for the [Regulations.gov APIs](https://www.regulations.gov/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Regulations.gov APIs](https://www.regulations.gov/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://www.regulations.gov/)- [Developer](https://www.regulations.gov/)- [GitHub](https://github.com/regulationsgov)The [APIs.json](https://github.com/api-evangelist/regulations-gov/blob/master/apis.json) index provides links to all the relevant links for the Regulations.gov APIs, including the OpenAPI and Postman if they are present.
